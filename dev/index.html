GeoIP

IP Geolocalization using the Geolite2 Database

Usage

GeoIP.geolocate(IPv4) will load data from the CSV if it's not already loaded.

Example

using GeoIP
a = ip"1.2.3.4"
geolocate(a)        # returns dictionary with all relevant information
